# kakaoChat
카카오 챗봇 스킬 서버 생성
<br><br><br>

## 사용 언어
- Node.js v22.11.0
- eexpress
- JavaScript
<br><br><br>

## 기능
### 1. 오늘 뭐 먹지?
  - 서버가 요청한 Request 중, user가 입력한 키워드를 추출
  - 추출한 키워드가 json 데이터 파일에 존재할 경우 random 함수를 이용해 무작위로 한개의 음식을 추출
  - json 데이터 형식으로 서버에 응답
